## 사용자 스토리

> 사용자는 헤더의 뷰 전환 버튼을 클릭해 리스트 뷰와 칸반 보드 뷰를 전환할 수 있다.

---

## 구현 세부사항

**상태 추가**
```ts
const [view, setView] = useState<'list' | 'kanban'>('list')
```

**뷰 토글 버튼 위치**  
헤더 우측, "새 할 일" 버튼 왼쪽에 배치

**버튼 UI**
- 리스트 뷰 아이콘: lucide `List`
- 칸반 뷰 아이콘: lucide `Columns`
- 현재 활성 뷰: `bg-zinc-800 text-zinc-100`
- 비활성: `text-zinc-400 hover:text-zinc-200`

**렌더링 분기**
```tsx
{view === 'list' ? (
  <ul>...</ul>          // 기존 투두 목록
) : (
  <KanbanBoard ... />   // ES-S2에서 구현
)}
```

**필터 연동**  
사이드바 필터(`filter` 상태)는 두 뷰 공통 적용.  
칸반 뷰에서 필터 선택 시 해당 상태 컬럼의 카드만 표시.

---

## 완료 기준

- 뷰 전환 버튼이 헤더 우측에 렌더링된다
- 버튼 클릭 시 리스트 ↔ 칸반 뷰가 전환된다
- 활성 뷰 버튼이 하이라이트 처리된다
- 사이드바 필터가 두 뷰 모두에서 동작한다
